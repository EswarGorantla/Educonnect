Course Overview

Total Duration: 12 Weeks

Level: Beginner to Intermediate

Mode: Online + Live Projects

Weekly Plan

  • Week 1: Introduction to Markup Languages (HTML, XML, XHTML, WML), CSS, DHTML, Client vs. Server Technologies.
  • Week 2: Hyperlinks, Frames & Framesets, HTML Tables and Attributes.
  • Week 3: HTML Forms, form & input, CSS (Inline, Internal, External, Imported), Cascading Rules.
  • Week 4: CSS (IDs, Classes, Selectors), XHTML Syntax & Rules, DTD.
  • Week 5: XML (Trees, Namespaces), JavaScript DOM Basics.
  • Week 6: JavaScript DOM Nodes & Traversal, Event Handling Basics.
  • Week 7: JavaScript Events (onClick, onSubmit, onMouseOver, onMouseOut), Cookies.
  • Week 8: JSP Introduction, Comparison with Servlets, Architecture & Engines.
  • Week 9: JSP Lifecycle, Structure, Components, Tags, Small Web App.
  • Week 10: JSP Elements, Actions, Implicit Objects, Exception Handling.
  • Week 11: JSP Session Tracking, MySQL Setup, JSP-Database Integration.
  • Week 12: JDBC CRUD Operations, JDBC Architecture, Servlet Basics (doGet, doPost).
